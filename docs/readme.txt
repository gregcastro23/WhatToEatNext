# What To Eat Next

An intelligent food recommendation system that combines traditional culinary wisdom with modern nutritional science. Built by Gregory Castro, this application provides personalized meal suggestions based on time of day, season, and personal preferences.

## Features

- Real-time meal recommendations
- Multiple cuisine options (Japanese, Mediterranean, French, Italian, etc.)
- Detailed nutritional information
- Ingredient swap suggestions
- Seasonal awareness
- Professional resume integration

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide Icons

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
