## ðŸ”§ Quick Fixes

### Unused Variables
```bash
# Find unused variables
yarn lint 2>&1 | grep "no-unused-vars"

# Prefix with underscore if intentional
const _unused = value;
```

### Import Issues
```bash
# Check import resolution
yarn lint:import-resolution-test

# Fix path aliases in tsconfig.json
```

### Type Safety
```bash
# Find all 'any' usage
yarn lint 2>&1 | grep "no-explicit-any"

# Replace with proper type or unknown
const value: unknown = getData();
```
