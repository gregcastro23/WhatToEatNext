# What To Eat Next 🍲✨

An astrologically-influenced food recommendation application that suggests what to eat based on current planetary positions, lunar phases, and elemental alignments.

## Features

- **Today's Food Recommendation**: Get a quick, personalized food suggestion based on current astrological conditions
- **Food Recommendations**: Browse detailed food recommendations by category
- **Cuisine Recommendations**: Get suggestions for entire cuisines that align with your current astrological state
- **Cooking Methods**: Learn which cooking techniques are most beneficial based on planetary alignments

## How It Works

This application combines ancient wisdom from astrology, alchemy, and culinary arts with modern nutritional science to provide personalized food recommendations. The system takes into account:

- Current planetary positions
- Lunar phase
- Zodiac influences
- Planetary hours
- Elemental properties of foods
- Chakra influences
- Seasonal factors

## Getting Started

### Prerequisites

- Node.js (14.x or higher)
- Yarn package manager

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/what-to-eat-next.git
   cd what-to-eat-next
   ```

2. Install dependencies:
   ```
   yarn install
   ```

3. Start the development server:
   ```
   yarn dev
   ```
   
   Or use the included script:
   ```
   ./start-app.sh
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

## Technology Stack

- Next.js - React framework
- TypeScript - Type-safe JavaScript
- Tailwind CSS - Utility-first CSS framework
- Material-UI - React component library
- Various astrological and astronomical calculation libraries

## Usage

1. Visit the home page to see your personalized food recommendation for today
2. Browse through the Food Recommendations section to explore foods by category
3. Check out the Cuisine Recommendations for broader meal ideas
4. Learn about appropriate Cooking Methods based on current astrological conditions
