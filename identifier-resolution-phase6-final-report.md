```bash
sed -i '' 's/campaigns\.\[/campaigns[/g'
sed -i '' 's/recommendations\.\[/recommendations[/g'
```

#### Pattern 2: Function Signature Corrections
```bash
# Fixed patterns like: test('name': any, async () => { → test('name', async () => {
sed -i '' "s/': any, async/', async/g"
sed -i '' "s/': any, () =>/', () =>/g"
```
