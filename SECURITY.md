# Security Policy

## Supported Versions

The following versions of WhatToEatNext are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.3.x   | ✅ Yes             |
| 2.2.x   | ❌ No              |
| < 2.2.0 | ❌ No              |

## Reporting a Vulnerability

We take the security of WhatToEatNext seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1.  **Do not open a public issue.**
2.  Email your findings to [INSERT EMAIL].
3.  Include details of the vulnerability, a proof of concept if possible, and any suggested fixes.

We will acknowledge your report within 48 hours and provide a timeline for resolution. We follow a responsible disclosure policy and ask that you do not disclose the vulnerability publicly until we have had a chance to address it.

Thank you for helping keep the alchemical culinary community safe! 🌟
